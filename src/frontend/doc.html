生成短链接 API

用于将长链接转换为短链接，支持自定义短链。

接口路径/api/create
请求方法POST

请求参数

参数名类型必填说明
longUrlstring是需要生成短链的原始长链接
customCodestring否自定义短链（6-12位字母数字，不能重复、不能包含敏感词）
expireDaysnumber否有效期（天），默认30天，最大365天

请求示例

{
  "longUrl": "https://www.example.com/abc",
  "customCode": "mycode123",
  "expireDays": 30
}

返回示例

{
  "status": 1,
  "shortUrl": "https://35url.cn/mycode123",
  "expire": 1718889600000,
}

status=1 表示成功，shortUrl为生成的短链，expire为到期时间戳（可选）。
